Задание 1-1: Системные вызовы
Описание задания: Создать файл numbers.txt с числами 1-100 (по строкам), подсчитать сумму, записать в sum.txt
Дополнительные требования:
• Используйте только системные вызовы: open(), read(), write(), close()
• НЕ используйте: fopen(), fprintf(), fscanf()
• Обрабатывайте ошибки с помощью perror()
• Компиляция: gcc -Wall -Wextra -o task task.c

Результат в sum.txt - Sum = 5050
