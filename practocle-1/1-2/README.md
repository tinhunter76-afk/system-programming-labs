ПРАКТИКА 1: СИСТЕМНЫЕ ВЫЗОВЫ, ПРОЦЕССЫ, ФАЙЛЫ
Задание 1-2: Процессы fork/exec
Описание задания: Родитель создаёт 3 дочерних процесса. Каждый выводит свой PID и номер (1,2,3). Родитель ждёт всех и выводит "Все дочерние процессы завершены"
Требования:
Используйте: fork(), exec(), wait(), getpid(), getppid()
Обрабатывайте ошибки
Компиляция: gcc -Wall -Wextra -o task task.c

Результат вывода:

Дочерний процесс 1: PID=2314, PPID=2300
Процесс завершён
Дочерний процесс 2: PID=2315, PPID=2300
Процесс завершён
Дочерний процесс 3: PID=2316, PPID=2300
Процесс завершён
Все дочерние процессы завершены
